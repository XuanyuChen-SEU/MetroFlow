# MetroFlow 快速启动

本文档提供基于 CSV 直读模式的最短启动路径，不涉及 MySQL 配置。

## 1. 环境要求

- Python 3.10 及以上
- Node.js 18 及以上
- npm 9 及以上

## 2. 原始数据

运行前请确认以下文件存在：

- `data/count_df.csv`
- `data/metro_global.json`
- `data/station_info.json`

## 3. 生成处理后数据

在项目根目录执行：

```bash
python scripts/clean_data.py
```

成功后将生成：

- `data/processed/dim_station.csv`
- `data/processed/fact_hourly_flow.csv`
- `data/processed/macro_city_flow_clean.csv`
- `data/processed/shanghai_line_topology.json`

## 4. 启动后端

```bash
cd backend
pip install -r requirements.txt
python app.py
```

默认地址：

- `http://127.0.0.1:5000`

建议首先检查以下接口：

- `http://127.0.0.1:5000/api/overview`
- `http://127.0.0.1:5000/api/overview/cities`
- `http://127.0.0.1:5000/api/shanghai/topology`

## 5. 启动前端

```bash
cd frontend
npm install
npm run dev
```

默认地址：

- `http://127.0.0.1:5173`

前端通过 Vite 代理访问后端接口，无需额外修改请求地址。

## 6. 使用 demo.html

如需快速联调，可在后端启动后直接打开项目根目录下的 `demo.html`。

如浏览器限制本地文件访问，建议在项目根目录执行：

```bash
python -m http.server 8080
```

访问地址：

- `http://127.0.0.1:8080/demo.html`

## 7. 推荐顺序

```text
1. python scripts/clean_data.py
2. cd backend && python app.py
3. 打开 demo.html 或启动 Vue 前端
4. cd frontend && npm run dev
```
